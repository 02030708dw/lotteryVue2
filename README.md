# GameClient

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install or yarn

# php serve localhost:8000
npm run server

# proxt serve with hot reload at localhost:8084
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
